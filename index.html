Apa itu Fungsi Kuadrat?

Fungsi kuadrat adalah persamaan matematika berbentuk f(x) = ax² + bx + c, di mana a, b, dan c adalah konstanta, serta a ≠ 0. Grafik fungsi ini berbentuk parabola, yang:

  • Titik Puncak (Vertex): Titik tertinggi atau terendah dari parabola.
  • Sumbu Simetri: Garis vertikal yang membagi parabola menjadi dua bagian simetris.
  • Akar-Akar: Nilai x yang membuat f(x) = 0.

Fungsi kuadrat sering digunakan dalam berbagai bidang, seperti fisika, ekonomi, dan teknik untuk memodelkan peristiwa berbentuk parabola.

Materi Fungsi Kuadrat

Pengertian

Fungsi kuadrat adalah fungsi matematika berbentuk polinomial yang memiliki variabel berpangkat dua sebagai pangkat tertingginya. Bentuk umum dari fungsi kuadrat adalah:

f(x) = ax² + bx + c

Di mana:

  • a, b, c: Konstanta (a ≠ 0)
  • x: Variabel bebas

Ciri-Ciri Fungsi Kuadrat

  • Bentuk Parabola: Grafik fungsi kuadrat selalu berupa parabola. Jika a > 0, parabola terbuka ke atas; jika a < 0, parabola terbuka ke bawah.
  • Titik Puncak: Titik tertinggi atau terendah dari parabola.
  • Sumbu Simetri: Garis vertikal yang membagi parabola menjadi dua bagian yang simetris.
  • Akar-Akar: Nilai x yang membuat f(x) = 0 (jika ada).
  • Titik Potong Sumbu Y: Titik di mana parabola memotong sumbu y, yaitu f(0) = c.

Rumus Penting

  • Titik Puncak: ( x = -b / 2a )
  • Diskriminan: ( D = b² - 4ac )
  • Rumus Kuadrat: ( x = (-b ± √D) / 2a )

Contoh Soal dan Penyelesaian Fungsi Kuadrat

Selesaikan persamaan kuadrat berikut:

f(x) = x² + 3x + 2

Penyelesaian

  1. Hitung diskriminan (D):

    D = b² - 4ac

    D = 3² - 4 * 1 * 2

    D = 9 - 8 = 1

  2. Titik balik (vertex):

    x = -b / 2a = -3 / (2 * 1) = -1.50

    y = f(-1.5) = (-1.5)² + 3(-1.5) + 2 = -0.25

    Titik balik adalah (-1.50, -0.25)

  3. Cari akar-akar persamaan:
    • x₁ = (-b + √D) / 2a = (-3 + √1) / 2 = -1.00
    • x₂ = (-b - √D) / 2a = (-3 - √1) / 2 = -2.00

Kesimpulan: Akar-akar persamaan adalah x₁ = -1.00 dan x₂ = -2.00.

Koordinat: (x: -, y: -)
